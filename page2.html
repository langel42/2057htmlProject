Web Technologies: Understanding Protocols and Standards

Web Technologies

Technology Description
HTML Hypertext Markup Language for creating web pages.
CSS Cascading Style Sheets for styling web pages.
JavaScript Client-side scripting language for dynamic web content.

Types of Protocols

Protocol Description
HTTP Hypertext Transfer Protocol for transmitting data over the web.
TCP/IP Transmission Control Protocol/Internet Protocol is a suite of communication protocols used to interconnect network devices on the internet.
SMTP Simple Mail Transfer Protocol is a protocol used for sending email messages between servers.
PPP Point-to-Point Protocol is a data link protocol used to establish a direct connection between two nodes.
FTP File Transfer Protocol is a standard network protocol used to transfer files between a client and a server on a computer network.
SFTP SSH File Transfer Protocol is a secure file transfer protocol that provides secure file access, transfer, and management functionalities over SSH.
HTTPS Hypertext Transfer Protocol Secure is an extension of HTTP used for secure communication over a computer network, especially the internet.
